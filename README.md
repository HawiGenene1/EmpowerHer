# EmpowerHer
Web-Based Platform for Empowering Women through Skills, Mentorship, and Community Support

## System Architecture

The system follows MVC architecture:

- **Frontend (View)**: React.js for dynamic and responsive UI.
- **Backend (Controller)**: Node.js with Express.js RESTful API.
- **Database (Model)**: PostgreSQL for structured and scalable relational data storage.
- **Authentication**: JWT (JSON Web Token) for secure login.
